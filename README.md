# fSoap
kSoap library for C# written in WinRT

这是一个用纯 Windows RT 写出来的soap库，用来在通用应用程序中使用。

本soap库完全照搬了[ksoap2-android][1]库，其中有很多不符合C#的写法存在，在后续的更新中会逐渐修正。
